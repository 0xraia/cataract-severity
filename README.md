# Cataract Severity

In Development

## Installation

```bash
pip install cataractseverity
```

## How To Use

```python
from cataractseverity import WindowCamera
import cv2

WindowCamera(cv2.VideoCapture(0), filename="output.jpg", width=640, height=480)
```
